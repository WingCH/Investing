# AI ScreenShot Toolkit

這是一個適用於 MetaTrader 4 (MT4) 的自動化截圖腳本，專門設計用於捕捉圖表特定區域（例如交易面板），方便進行記錄或 AI 分析。

## 功能特點 (Features)

1.  **高兼容性**: 移除了 "EA 名稱檢查"，支援舊版 MT4，確保在不同環境下均能運行。
2.  **定點截圖**: 專門捕捉圖表 **左側 (Left Side)** 區域，預設捕捉交易面板。
3.  **自定義尺寸**: 預設截圖大小為 **500x700 像素**，並可在腳本輸入參數中調整。
4.  **自動命名**:
    *   輸出路徑: `MQL4/Files/AI_Snapshots/YYYY.MM.DD_HH-MM/`
    *   檔案命名: `貨幣對_序號.png` (例如 `EURUSD_1.png`, `EURUSD_2.png`)
5.  **批量處理**: 一鍵運行，自動遍歷並截取當前終端機內 **所有** 打開的圖表。
6.  **完成提示**: 任務完成後會彈出視窗，詢問是否直接打開截圖文件夾。

## 安裝步驟 (Installation)

1.  打開 MT4 交易平台。
2.  點擊選單欄的 **文件 (File)** -> **打開數據文件夾 (Open Data Folder)**。
3.  進入 `MQL4` -> `Scripts` 文件夾。
4.  將 `AI_ScreenShot.mq4` 文檔複製到此目錄中。
5.  回到 MT4，在 **導航器 (Navigator)** 面板中，右鍵點擊 **腳本 (Scripts)** -> **刷新 (Refresh)**。
6.  現在你應該能看到 `AI_ScreenShot` 出現在腳本列表中。

## 使用方法 (Usage)

1.  確保你想要截圖的圖表已經打開。
2.  在導航器中雙擊 `AI_ScreenShot` 腳本，或將其拖拽到任意圖表上。
3.  (可選) 在彈出的參數視窗中調整 `ImageWidth` (寬度) 和 `ImageHeight` (高度)。
4.  點擊 **確定 (OK)**。
5.  腳本將自動切換並截取所有圖表。
6.  完成後，點擊彈窗中的 **Yes** 即可打開保存截圖的文件夾。

## 注意事項 (Notes)

*   **截圖區域**: 腳本使用的是 `ALIGN_LEFT`，這意味著它會從圖表視窗的左邊緣開始截取。請確保你要捕捉的內容（如面板）位於圖表左側。
*   **文件位置**: 截圖會保存在 MT4 數據文件夾下的 `MQL4\Files\AI_Snapshots` 目錄中，並按時間分文件夾整理。
